Before you play, connect your Firebase project for authentication, game state, and multiplayer.

  1. Go to console.firebase.google.com and create a project.
  2. Enable Authentication → Google sign-in method.
  3. Create a Firestore Database (test mode is fine to start).
  4. Create a Realtime Database for presence tracking.
  5. Register a Web App and copy the config object below.

The config is stored in your browser's localStorage.
