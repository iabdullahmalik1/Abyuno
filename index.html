📧 Email Notifications Setup Required
To enable email notifications (challenges, friend requests, welcome):
  1. Go to emailjs.com → Free account (200 emails/month)
  2. Create a Gmail service → Copy Service ID
  3. Create 4 email templates (IDs: template_challenge, template_welcome, template_friend_req, template_friend_acc)
  4. Copy your Public Key from Account → API Keys
  5. Replace YOUR_EMAILJS_PUBLIC_KEY and YOUR_SERVICE_ID in the JS
See the comment in the source for ready-to-paste HTML templates.
