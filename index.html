Play UNO Online Free — Abyuno Multiplayer Card Game

Abyuno is a free, browser-based multiplayer UNO card game that you can play instantly with 2 to 8 players. No download, no install — just open the website and start playing classic UNO online with friends.

How to Play UNO Online

Sign in with your Google account, create a private room or join a public lobby, share your 6-character room code with friends, and start playing. The game follows standard UNO rules: match cards by colour or number, use Wild cards to change colour, and call UNO when you have just 2 cards left.

UNO Card Game Rules

  • Each player starts with 7 cards
  • Match the top card by colour or number
  • Skip — next player loses their turn
  • Reverse — play direction flips
  • Draw Two (+2) — next player draws 2 cards
  • Wild — change the current colour
  • Wild Draw Four (+4) — change colour and next player draws 4
  • Call UNO when you have 2 cards left or draw a 5-card penalty

Features

  • Free multiplayer UNO online — no ads during gameplay
  • Private rooms with shareable invite links
  • Public lobbies to find games with strangers
  • Challenge friends even when they are offline
  • In-game chat with emoji reactions
  • Works on mobile (iOS, Android) and desktop (Windows, Mac, Linux)
  • 3 beautiful themes: Midnight, Ocean, Sunset
  • Google Sign-In — no password required

About Abyuno

Abyuno is a free, open-source multiplayer UNO card game built entirely in the browser using vanilla JavaScript and Firebase.

Tech Stack:
Frontend: Vanilla JS + HTML5 Canvas
Backend: Firebase Firestore + Realtime Database
Auth: Firebase Authentication (Google)
Email: EmailJS
Hosting: GitHub Pages

Features: Real-time multiplayer, friends system, challenge notifications, in-game chat, 3 themes, mobile-optimised, reconnect on disconnect.
