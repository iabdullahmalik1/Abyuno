## 🎯 2. Friend Challenges

### Sending a Challenge
1. Open **Friends Panel** (click 👥 icon, top right)
2. Go to **"Friends"** tab
3. Find the friend you want to challenge
4. Click **"Challenge"** button (replaces Invite when in lobby)
5. They receive a notification instantly

### Receiving a Challenge
1. Get a browser notification: "*[Friend] challenged you to UNO!*"
2. See counter badge on Friends icon showing pending challenges
3. Open Friends → "Requests" tab to see challenges
4. Click **"Accept"** to join their room OR **"Decline"** to refuse

### Key Points
- ✅ Can challenge friends even if they're offline
- ✅ Challenges expire after 24 hours
- ✅ Friends get browser notifications  
- ✅ Works on mobile and desktop
- ✅ One-click accept/decline

## 🔔 3. Notifications System

### Browser Notifications
**First Time Setup:**
1. Visit ABYUNO
2. You'll see a prompt: "*Enable notifications for challenges?*"
3. Click "Allow" to enable
4. Or later: Go to Browser Settings → Notifications → Allow ABYUNO

**What You'll Get Notified About:**
- Friend sends you a challenge
- Friend accepts your challenge
- Other players call UNO (during gameplay)
- Friend goes online after being challenged

### Notification Settings
- Notifications appear even if browser tab is closed
- Sound enabled by default (system beep)
- Can manage in browser status bar
- No data tracking - all local

## 👥 4. Friends System Improvements

### Full Friends Workflow
1. **Find a Friend**: Friends Tab → "Find" → Type exact username
2. **Send Request**: Click "Add" → appears as pending
3. **Accept/Decline**: Other person approves in "Requests" tab
4. **Gaming Together**: 
   - **Invite**: Click "Invite" during gameplay
   - **Challenge**: Click "Challenge" even when not in game

### Status Indicators
- 🟢 **Online** - Friend is on the site
- ⚫ **Offline** - Friend not on site currently  
- 🟡 **In Game** - Friend is playing now
- 🔵 **In Lobby** - Friend waiting for game to start

### Actions Available
- **Invite** (only in lobby): Add to current room
- **Challenge** (always): Send challenge invite
- **Remove**: Delete friend permanently (asks for confirmation)

### What's New

- ⚡ Red-purple flash across screen

- 📢 System chat message: "*[Player] called UNO!*"

- **Your Turn**: Blue glow around your avatar
- **Someone called UNO**: Screen flash effect

## 🎯 9. Troubleshooting

### "Friends not loading"
- ✓ Log out and back in
- ✓ Refresh the page
- ✓ Check internet connection

### "Theme looks wrong"
- ✓ Clear browser cache
- ✓ Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
- ✓ Try another theme then switch back

### "Challenge notification not appearing"
- ✓ Check browser notification settings
- ✓ Click "Allow" if prompted
- ✓ Look for challenge in Friends panel anyway

### "Cards overlapping"
- ✓ This is now prevented automatically
- ✓ If still happening, refresh page
- ✓ Check if elements are colliding with UI

### "UNO effects not showing for others"
- ✓ Check internet connection
- ✓ Others should see your effect automatically
- ✓ Reload if issue persists
